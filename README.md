# Mantom Landing

Mermer ve doğal taş ihracatı yapan Mantom için modern, animasyonlu landing page.

## Commands

```bash
npm install
npm run generate:assets
npm run dev
```

## Notes

- Sayfa Next.js App Router, GSAP, ScrollTrigger ve bölümlere ayrılmış komponent mimarisi kullanır.
- Ürün ve hero taş görselleri `public/stones` ve `public/` altında yerel gerçek fotoğraf JPG dosyalarıdır.
- Proje görselleri `public/projects` altında yerel gerçek fotoğraf JPG dosyalarıdır; kaynak linkleri `data/stones.ts` içindeki proje datasında tutulur.
